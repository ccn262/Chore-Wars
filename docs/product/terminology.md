# Terminology

Use consistent product language across the repo and app.

## Core terms

- **Household**: the group sharing chores and points
- **Member**: a person in a household
- **Chore**: a task that needs to be done
- **Completion**: the act of marking a chore done
- **Points**: the reward value assigned to a chore
- **Balance**: the current score or total points for a member
- **Cycle**: the repeat pattern for a chore
- **House rules**: shared settings for the household

## Naming guidance

- Prefer plain language over internal jargon
- Avoid jargon that makes the app feel technical
- Keep labels short enough for small screens
- Use language that translates cleanly
