# Seed Strategy

## Seed goals

- Provide predictable starter data for local development
- Demonstrate the main household and chore flows
- Avoid leaking real user data
- Seed starter chore categories and chore templates
- Include sample households, members, chores, and completions
- Keep seed data fake, reset-friendly, and easy to replace later

## Seed contents

- One or more sample households
- Sample members
- Sample chores
- Sample completions
- Sample point totals
- Starter chore categories
- Starter chore templates

## Seed rules

- Seeds should be easy to reset
- Keep them clearly fake
- Make them useful for UI preview and testing
- Keep names and content locale-ready where possible
- Avoid hardcoding assumptions about currency or region
