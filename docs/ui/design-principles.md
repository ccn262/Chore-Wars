# Design Principles

## Visual direction

- Mobile-first layout
- App-like full-screen experience
- Minimal, clear, colourful, and uncluttered
- Playful but polished
- Built to feel like a phone app first
- Bottom navigation preferred
- Big rounded buttons
- Icon-first chore cards

## Layout principles

- Home screen carries the main story
- Use cards, chips, and large tap targets
- Keep the primary action visible without hunting
- Avoid dense dashboards and desktop-like panels
- Prioritise fast scanning and instant feedback
- Keep reports card-based, chart-led, badge-supported, and easy to understand

## Interaction principles

- Thumb-first controls
- Quick tap interactions
- Clear hierarchy
- Immediate feedback after actions
- Minimal text
- Playful animations used sparingly
- One-tap chore completion should be the default path

## Accessibility and localisation

- Use readable contrast
- Design for text expansion
- Keep copy short and flexible
- Support locale-aware formatting by default
