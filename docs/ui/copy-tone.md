# Copy Tone

## Voice

- Friendly
- Playful
- Brief
- Polished
- Clear

## Tone rules

- Keep humour light and optional
- Avoid sarcasm that could alienate users
- Avoid childish copy
- Write like a helpful household assistant, not a game announcer

## Microcopy guidance

- Prefer short labels over long explanations
- Use action-led language
- Keep translations in mind when drafting text
