# Screen Inventory

## Core screens

- Home
- Household setup
- Chore detail
- Quick complete sheet
- Reporting summary
- Member profile
- Settings
- Onboarding

## Reporting surfaces

- Progress cards
- Streak or momentum cards
- Household summary cards
- Recent activity feed

## Screen priorities

- The home screen should be the most polished screen
- Reporting should be visual and easy to scan
- Settings should stay simple and secondary
