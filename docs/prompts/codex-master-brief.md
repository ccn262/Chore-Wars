# Codex Master Brief

Use this repo as a planning-first workspace for Chore Wars.

## Non-negotiables

- Keep the product mobile-first
- Optimise for thumb-first, quick tap interactions
- Make the home screen the hero screen
- Preserve a 3-second path to chore completion and points
- Keep the UI minimal, colourful, and uncluttered
- Keep the product playful but polished
- Support international users from the start

## Current stack direction

- Next.js
- Supabase
- Vercel
- Resend
- GitHub
- Porkbun
- Expo later

## Task guidance

- Read the relevant docs before changing anything
- Update logs when decisions matter
- Avoid implementation work unless explicitly asked
- Prefer the smallest safe, reviewable output
